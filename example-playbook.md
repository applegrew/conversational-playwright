# Example Automation Playbook

This is an example playbook that demonstrates automated browser testing steps.

## Steps to Execute

1. Navigate to https://www.google.com
2. Type "playwright automation" in the search box
3. Click the search button
4. Wait for search results to load
5. Click on the first search result

## Alternative Format

You can also use bullet points:

- Navigate to https://www.example.com
- Click on the "More information" link
- Scroll down to the footer
- Take a screenshot

Or simple lines (any non-heading, non-empty line will be treated as a step).
